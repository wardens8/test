## 简介

TypeScript是具有类型语法的JavaScript，是一种基于JavaScript构建的强类型编程语言，可为你提供任何规模的更好工具。TypeScript属于JavaScript的超集，可以兼容JavaScript的所有语法

## 安装

1. TypeScript是基于nodejs的，所以需要先安装nodejs
2. 全局安装TypeScript：`npm i -g typescript`
3. 创建一个ts文件：`index.ts`
4. 使用tsc对ts文件进行编译成为js文件：`tsc xxx.ts`

## 类型声明

类型声明是TypeScript非常重要的一个特点，通过类型声明可以指定TypeScript中变量（参数，形参）的类型。指定类型后，当为变量赋值时，TypeScript编译器会自动检查值是否符合类型声明，符合则赋值，否则报错

简而言之，类型声明给变量设置了类型，使得变量智能存储某种类型的值

- 给变量设置类型

```typescript
let 变量: 类型
let 变量: 类型 = 值
```

- 给函数的参数设置类型

```typescript
function fn(参数: 类型, 参数: 类型){
  ...
}
```

- 给函数的返回值设置类型

```typescript
function fn(): 类型{
  return ...
}
```

- TypeScript自动判断类型
  - TypeScript拥有自动的类型判断机制，当对变量的声明和复制是同时进行的，TypeScript编译器会自动判断变量的类型，所以如果你的变量的声明和赋值是同时进行的，可以忽略掉类型声明

```typescript
let a = 1
// 自动进行类型判断，相当于let a:number = 1;
```
