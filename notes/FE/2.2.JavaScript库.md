## jQuery 基础

jQuery 顶级对象 `$`

- `$` 是jQuery的别称，在代码中可以使用jQuery代替`$`，但一般为了方便，通常都直接使用`$`
- `$` 是jQuery的顶级对象，相当于原生JavaScript中的window。把元素利用`$`包装成Query对象，就可以调用jQuery的方法

jQuery 对象和 DOM 对象

- DOM对象：通过原生JavaScript获取页面中的元素返回的对象
- jQuery对象：通过jQuery获取页面中的元素返回的对象
- DOM对象转jQuery对象：`$(DOM对象)`
- jQuery对象转DOM对象：`jQuery对象[index]` 或 `jQuery对象.get(index)`

```javascript
// DOM对象
const div = document.querySelector('div')
// jQuery对象
$('span')

// DOM对象转变为jQuery对象
$(div)
// jQuery对象转变为DOM对象
$('span')[0]
$('span').get(0)
```

jQuery 入口函数：相当于原生的 `DOMContentLoaded`

- `$(document).ready(function() {})`：当文档加载完毕之后，自动执行该函数
- `$(function() {})`：当文档加载完毕之后，自动执行该函数

```javascript
$(document).ready(function () {
  // 此处是页面DOM加载完毕后执行的代码
})

$(function () {
  // 此处是页面DOM加载完毕后执行的代码
})
```

jQuery 中的 `this` 指向

- 原生JavaScript中，`this` 的指向跟随函数的调用方式不同而不同
- jQuery中，`this` 指向的是当前正在操作的 DOM 元素

jQuery 链式操作

- 链式操作：将多个 jQuery 方法用 `.` 连接起来，可以连续对同一个元素进行多个操作
- jQuery 中的大部分方法，都可以连缀调用，原因是 jQuery 中大部分方法返回的都是 jQuery 对象

```javascript
$('div').css('color', 'red').attr('title', 'hello').addClass('box')
```

## jQuery 选择器

jQuery 选择器用法与原生选择器用法相差不大，只是用 `$(...)` 代替了 `document`，并且返回的是 jQuery 对象，而不是 DOM 对象，原生的选择器基本都可以在 jQuery 中使用。例如下列基本选择器

- 基础选择器
  - `$('#id')` id选择器
  - `$('.class')` 类选择器
  - `$('div')` 元素选择器
  - `$('*')` 通配符选择器
- 复合选择器
  - `$('div, p, span')` 并集选择器
  - `$('div.class')` 交集选择器
  - `$('div p')` 后代选择器
  - `$('div > p')` 子元素选择器
  - `$('div ~ p')` 普通兄弟选择器
  - `$('div + p')` 相邻兄弟选择器
- 属性选择器
  - `$('div[attribute]')` 选择含有 attribute 属性的元素
  - `$('div[attribute="abc"]')` 选择含有 attribute 属性且属性值为 abc 的元素
  - `$('div[attribute!="abc"]')` 选择含有 attribute 属性且属性值不为 abc 的元素
  - `$('div[attribute^="a"]')` 选择含有 attribute 属性且属性值以 a 开头的元素
  - `$('div[attribute$="a"]')` 选择含有 attribute 属性且属性值以 a 结尾的元素
  - `$('div[attribute*="a"]')` 选择含有 attribute 属性且属性值包含 a 的元素
  - `$('div[attribute|="a"]')` 选择含有 attribute 属性且属性值为 a 或以 a- 开头的元素
- 筛选选择器
  - `$('div:first')` 选中第一个元素
  - `$('div:last')` 选中最后一个元素
  - `$('div:eq(index)')` 选中索引为 index 的元素
  - `$('div:odd')` 选中索引为奇数的元素
  - `$('div:even')` 选中索引为偶数的元素
- 表单选择器
  - `$('input:enabled')` 选中所有可用的元素
  - `$('input:disabled')` 选中所有不可用的元素
  - `$('input:checked')` 选中所有被选中的元素
  - `$('input:selected')` 选中所有被选中的元素
- 选择器的方法
  - `parent()`/`$('li').parent()` 查找父级元素
  - `children(selector)`/`$('ul').children('li')` 查找子元素，相当于$('ul>li')，最近一级（亲儿子）
  - `find(selector)`/`$('ul').find('li')` 查找后代元素，相当于$('ul li')，后代选择器
  - `siblings(selector)`/`$('.first').siblings('li')` 查找兄弟元素，不包括自己本身
  - `nextAll()`/`$('.first').nextAll()` 查找当前元素之后所有的同辈元素
  - `prevtAll()`/`$('.last').prevAll()` 查找当前元素之前所有的同辈元素
  - `eq(index)`/`$('li').eq(2)` 选中索引为 2 的元素，相当于$('li:eq(2)')

> jQuery 有隐式迭代，所以不需要使用循环遍历

## jQuery 属性

属性

- `prop('property, ['value'])` 设置或返回匹配元素的属性和值（DOM属性）
- `removeProp('property')` 从所有匹配的元素中移除指定的属性
- `attr('attribute', ['value'])` 设置或返回匹配元素的属性和值（标签属性）
- `removeAttr('attribute')` 从所有匹配的元素中移除指定的属性
- `data('key', ['value'])` 设置或返回匹配元素的 data 属性和值，属性存到内存内

```javascript
// 获取属性
$('div').prop('title')
$('div').prop('title', 'hello')
// 移除属性
$('div').removeProp('title')

// 获取属性
$('div').attr('title')
$('div').attr('title', 'hello')
// 移除属性
$('div').removeAttr('title')

// 获取data属性
$('div').data('title')
$('div').data('title', 'hello')
$('div').data('index') // 获取h5自定义属性data-index
```

> 二者的区别：prop 是针对 DOM 属性，attr 是针对标签属性，对于布尔类型的属性，如 checked、selected、disabled，用 prop 更好，对于其他属性，attr 更好

类

- `hasClass(className1 className2 ...)` 检查匹配的元素是否拥有指定的类，可用字符串和数组
- `addClass(className1 className2 ...)` 向匹配的元素添加指定的类名，可用字符串和数组
- `removeClass([className1 className2 ...])` 从所有匹配的元素中删除全部或者指定的类，可用字符串和数组，不传参数则移除所有类
- `toggleClass(className1 className2 ...)` 从匹配的元素中添加或删除一个类，可用字符串和数组，如果存在则删除，不存在则添加

```javascript
$('div').hasClass('box')
$('div').hasClass('box box1 box2')
$('div').hasClass(['box', 'box1', 'box2'])

$('div').addClass('box')
$('div').addClass('box box1 box2')
$('div').addClass(['box', 'box1', 'box2'])

$('div').removeClass() // 移除所有类
$('div').removeClass('box')
$('div').removeClass('box box1 box2')
$('div').removeClass(['box', 'box1', 'box2'])

$('div').toggleClass('box')
$('div').toggleClass('box box1 box2')
$('div').toggleClass(['box', 'box1', 'box2'])
```

文本值

- `html('...')` 设置或返回匹配的元素集合中的HTML内容（可以写html内容）
- `text('...')` 设置或返回匹配元素的内容（不可以写html内容）
- `val('...')` 设置或返回匹配元素的值（多用于input元素）

```javascript
$('div').html() // 获取html内容
$('div').html('<h1>hello</h1>') // 设置html内容

$('div').text() // 获取文本内容
$('div').text('<h1>hello</h1>') // 设置文本内容

$('input').val() // 获取input的值
$('input').val('hello') // 设置input的值
```

## jQuery CSS

CSS

- `css('attribute', 'value')` 设置或返回匹配元素的样式属性
- `css({ 'attribute': 'value' })` 设置或返回匹配元素的样式属性

```javascript
$('div').css('width')
$('div').css('width', '100px')
$('div').css('width', '100px').css('height', '100px')
$('div').css({ width: '100px', height: '100px' })
```

尺寸

- `width()`/`height()` 设置或返回匹配元素的宽度/高度
- `innerWidth()`/`innerHeight()` 设置或返回匹配元素的宽度/高度（+padding）
- `outerWidth()`/`outerHeight()` 设置或返回匹配元素的宽度/高度（+padding+border）
- `outerWidth(true)`/`outerHeight(true)` 设置或返回匹配元素的宽度/高度（+padding+border+margin）

```javascript
// 获取宽度和高度
$('div').width()
$('div').height()
// 设置宽度和高度
$('div').width(300)
$('div').height(300)

// 获取宽度和高度（+padding）
$('div').innerWidth()
$('div').innerHeight()
// 设置宽度和高度
$('div').innerWidth(300)
$('div').innerHeight(300)

// 获取宽度和高度（+padding+border）
$('div').outerWidth()
$('div').outerHeight()
// 设置宽度和高度
$('div').outerWidth(300)
$('div').outerHeight(300)

// 获取宽度和高度（+padding+border+margin）
$('div').outerWidth(true)
$('div').outerHeight(true)
// 设置宽度和高度
$('div').outerWidth(true, 300)
$('div').outerHeight(true, 300)
```

位置

- `offset()` 返回第一个匹配元素相对于文档的位置
- `offsetParent()` 返回最近的定位祖先元素
- `position()` 返回第一个匹配元素相对于父元素的位置
- `scrollTop()` 设置或返回匹配元素相对滚动条顶部的偏移
- `scrollLeft()` 设置或返回匹配元素相对滚动条左侧的偏移

```javascript
$('div').offset() // 获取元素相对于文档的位置
$('div').offset().top // 获取元素相对于文档的顶部位置
$('div').offset().left // 获取元素相对于文档的左侧位置
$('div').offset({ top: 100, left: 100 }) // 设置元素相对于文档的位置

$('div').offsetParent() // 获取最近的定位祖先元素

$('div').position() // 获取元素相对于父元素的位置
$('div').position().top // 获取元素相对于父元素的顶部位置
$('div').position().left // 获取元素相对于父元素的左侧位置

$(document).scrollTop() // 获取文档相对滚动条顶部的偏移
$(document).scrollTop(100) // 设置文档相对滚动条顶部的偏移

$(document).scrollLeft() // 获取文档相对滚动条左侧的偏移
$(document).scrollLeft(100) // 设置文档相对滚动条左侧的偏移
```

## jQuery 元素操作

遍历

- `each(function(index, DOMElement))` 遍历匹配的元素集合
- `$.each(object, function(index, DOMElement))` 遍历一个对象或数组

```javascript
$('div').each(function (index, DOMElement) {
  console.log(index, DOMElement)
})

$.each({ name: 'hello', age: 18 }, function (index, value) {
  console.log(index, value)
})
```

创建

- `$('html')` 创建元素

```javascript
$('<li>我是新创建的</li>')
const li = $('<li>我是新创建的</li>')
```

添加

- `append(content, [content])` 元素结尾插入指定内容
- `prepend(content, [content])` 元素开头插入指定内容
- `appendTo(content)` 将元素插入至指定内容结尾
- `prependTo(content)` 将元素插入至指定内容开头
- `before(content, [content])` 在元素之前插入指定内容
- `after(content, [content])` 在元素之后插入指定内容
- `insertBefore(content)` 将元素插入到指定内容之前
- `insertAfter(content)` 将元素插入到指定内容之后

```javascript
$('div').append('<p>hello</p>') // 向div中追加内容
$('div').prepend('<p>hello</p>') // 向div中前置内容

$('p').appendTo('div') // 把p追加到div中
$('p').prependTo('div') // 把p前置到div中

$('div').before('<p>hello</p>') // 在div之前插入内容
$('div').after('<p>hello</p>') // 在div之后插入内容

$('p').insertBefore('div') // 把p插入到div之前
$('p').insertAfter('div') // 把p插入到div之后
```

删除

- `empty()` 从匹配的元素集合中删除所有的子节点和内容
- `remove([selector])` 从 DOM 中删除所有匹配的元素
- `detach([selector])` 从 DOM 中删除所有匹配的元素，保留数据和事件

```javascript
$('div').empty() // 清空div内容

$('div').remove() // 删除div
$('div').remove('span') // 删除div中的span

$('div').detach() // 删除div
$('div').detach('span') // 删除div中的span
```

复制

- `clone()` 创建匹配元素集合的副本

```javascript
$('div').clone()
$('div').clone().appendTo('body')
```

替换

- `replaceWith(content)` 用指定内容替换所有匹配元素
- `replaceAll(selector)` 用匹配元素替换所有匹配的选择器

```javascript
$('div').replaceWith('<p>hello</p>') // 用p替换div
$('<p>hello</p>').replaceAll('div') // 用p替换div
```

包裹

- `wrap(html)` 把每个匹配的元素用指定的HTML元素结构包裹起来
- `unwrap()` 把匹配元素集合的父级元素删除，保留自身（解除包裹）
- `wrapAll(html)` 把所有匹配的元素放在一起用一个指定的HTML元素结构包裹起来
- `wrapInner(html)` 把每个匹配的元素的子内容用指定的HTML结构包裹起来

```javascript
$('p').wrap("<div class='wrap'></div>") // 每个p外用div包裹起来
$('p').unwrap() // 解除p的外包裹
$('p').wrapAll('<div></div>') // 所有p外用div包裹起来
$('p').wrapInner('<b></b>') // 每个p的子内容用b包裹起来
```

## jQuery 事件绑定

事件基础绑定

- `click(fn)` 单击事件
- `dblclick(fn)` 双击事件
- `mousedown(fn])` 鼠标按下事件
- `mouseup(fn)` 鼠标弹起事件
- `mousemove(fn)` 鼠标移动事件
- `mouseover(fn)` 鼠标经过事件
- `mouseout(fn)` 鼠标离开事件
- `mouseenter(fn)` 鼠标进入事件
- `mouseleave(fn)` 鼠标离开事件
- `keyup(fn)` 键盘弹起事件
- `keydown(fn)` 键盘按下事件
- `keypress(fn)` 键盘按下并弹起事件
- blur([[data],fn])
- change([[data],fn])
- error([[data],fn])
- focus([[data],fn])
- focusin([data],fn)
- focusout([data],fn)
- resize([[data],fn])
- scroll([[data],fn])
- select([[data],fn])
- submit([[data],fn])

```javascript
$('div').click(function () {
  console.log('单击事件')
})
```

> 其他的事件方法同理，只需要把 click 换成对应的事件名即可

事件通用绑定

- `on(events, [selector], [data], fn)` 绑定一个或多个事件到所选元素上，触发时执行函数

```javascript
// 事件绑定单个事件
$('div').on('click', function () {
  console.log('单击事件')
})
// 事件绑定多个事件
$('div').on('click mousemove', function () {
  console.log('单击事件或鼠标移动事件')
})
// 事件绑定多个事件，多个事件绑定同一个函数
$('div').on({
  click: function () {
    console.log('单击事件')
  },
  mousemove: function () {
    console.log('鼠标移动事件')
  },
})
// 事件委托，给父元素绑定事件，子元素触发事件
$('div').on('click', 'span', function () {
  console.log('单击span事件')
})
```

- `off(events, [selector], [data], fn)` 移除所绑定的事件

```javascript
$('div').off() // 移除所有事件
$('div').off('click') // 移除单击事件
$('div').off('click mousemove') // 移除单击事件和鼠标移动事件
$('div').off('click', 'span') // 移除事件委托，单击span事件
```

- `one(events, [data], fn)` 为元素绑定一个或多个事件，只触发一次

```javascript
$('div').one('click', function () {
  console.log('单击事件')
})
```

- `$(element).[events]()` 自动触发单击事件
- `trigger(events, [data])` 根据事件类型触发所绑定的事件
- `triggerHandler(events, [data])` 根据事件类型触发所绑定的事件，只触发第一个匹配元素的事件处理函数，并立即返回执行结果，不触发浏览器的默认行为，不冒泡

```javascript
$('div').click() // 自动触发单击事件
$('div').trigger('click') // 根据事件类型触发所绑定的事件
$('div').triggerHandler('click') // 根据事件类型触发所绑定的事件，只触发第一个匹配元素的事件处理函数，并立即返回执行结果，不触发浏览器的默认行为，不冒泡
```

事件切换

- `hover([mouseenter,] mouseleave)` 鼠标经过和离开，第一个函数是经过，第二个函数是离开，如果只写一个函数，则鼠标经过和离开都会触发这个函数

```javascript
$('div').hover(
  function () {
    console.log('鼠标经过')
  },
  function () {
    console.log('鼠标离开')
  },
)

$('div').hover(function () {
  console.log('鼠标经过或离开')
})
```

event 事件对象：jQuery 中的事件对象与原生 JavaScript 中的事件对象几乎完全相同，都是用来记录事件的信息和属性，正常使用即可

## jQuery 动画效果

显示隐藏

- `show([speed], [easing], [fn])` 显示
- `hide([speed], [easing], [fn])` 隐藏
- `toggle([speed], [easing], [fn])` 切换显示/隐藏

```javascript
$('div').show(100)
$('div').hide(100)
$('div').toggle(100)
```

滑动

- `slideDown([speed], [easing], [fn])` 向下展开
- `slideUp([speed], [easing], [fn])` 向上卷起
- `slideToggle([speed], [easing], [fn])` 切换展开/卷起

```javascript
$('div').slideDown(100)
$('div').slideUp(100)
$('div').slideToggle(100)
```

淡入淡出

- `fadeIn([speed], [easing], [fn])` 淡入
- `fadeOut([speed], [easing], [fn])` 淡出
- `fadeToggle([speed], [easing], [fn])` 切换淡入/淡出
- `fadeTo([speed], opacity, [easing], [fn])` 以淡入淡出的方式调整到指定的透明度

```javascript
$('div').fadeIn(100)
$('div').fadeOut(100)
$('div').fadeToggle(100)
$('div').fadeTo(100, 0.5) // 透明度
```

自定义动画

- `animate(params, [speed], [easing], [fn])` 执行自定义动画
- `stop()` 停止当前正在运行的动画，然后开始新的动画
- `delay(d,[q])`
- `finish([queue])`

```javascript
$('div').animate({ width: '100px', height: '100px' }, 1000)
$('div').stop()
$('div').delay(1000)
$('div').finish()
```

> 只有元素才能做动画，document和window都不能做动画，需要页面滚动时选择html元素和body元素

## jQuery AJAX

- `$.ajax({ url, type, data, dataType, success, error, complete })` 全局AJAX事件
- `$.get(url, data, success, dataType)` GET请求专用
- `$.post(url, data, success, dataType)` POST请求专用
- `$.getJSON(url, data, success)` 专门请求JSON数据
- `load(url, data, callback)` 直接将请求结果加载到指定DOM元素中
- `ajaxStart()`、`ajaxStop()`、`ajaxSuccess()`、`ajaxError()` 等，用于全局监控 AJAX 状态

```javascript
// 全局AJAX事件
$.ajax({
  url: 'http://www.liulongbin.top:3006/api/getbooks',
  type: 'GET',
```

## jQuery 对象拷贝

- `$.extend([deep], target, object1, [objectN])` 拷贝对象
  - `deep` 可选。默认false为浅拷贝，true为深拷贝
  - `target` 目标对象
  - `object1` 源对象

```javascript
var obj1 = { name: 'zs', age: 18 }
var obj2 = { name: 'ls', age: 20 }
$.extend(true, obj1, obj2)
console.log(obj1) // { name: 'ls', age: 20 }
```

## jQuery 多库共存

- 由于 jQuery 的 `$` 符号可能被其他库占用，导致 jQuery 无法正常使用，因此可以释放 `$` 符号或 `jQuery` 关键字的控制权改为其他，让其他库可以使用 `$` 符号
- `jQuery.xxx()` 使用 `jQuery` 关键字代替 `$` 符号
- `$.noConflict()` 释放 `$` 控制权，把控制权交给其他库
- `jQuery.noConflict()` 释放 `jQuery` 控制权，把控制权交给其他库

```javascript
var aaa = $.noConflict() // 释放$控制权，用aaa代替，把控制权交给其他库
var bbb = jQuery.noConflict() // 释放jQuery控制权，用bbb代替，把控制权交给其他库
```

## Dayjs 使用

日期初始化

```javascript
// 获取当前时间
dayjs()
// 获取指定时间
dayjs('2020-01-01') // 字符串
dayjs('2020-01-01 00:00:00') // 字符串

// 原生
dayjs(new Date())
```

日期格式化

- 核心方法：`dayjs().format(formatStr)`
- 年：`YY` 两位数的年份，`YYYY` 四位数的年份
- 月：`M` 月份，从 1 开始，`MM` 月份，两位数，从 01 开始，`MMM` 简写的英文月份名称，`MMMM` 完整的英文月份名称
- 日：`D` 月份中的第几天，从 1 开始，`DD` 月份中的第几天，两位数，从 01 开始
- 时：`H` 小时，24 小时制，从 0 开始，`HH` 小时，24 小时制，两位数，从 00 开始，`h` 小时，12 小时制，从 1 开始，`hh` 小时，12 小时制，两位数，从 01 开始
- 星期：`ddd` 星期，英文缩写，`dddd` 星期，英文全称
- 分：`m` 分钟，从 0 开始，`mm` 分钟，两位数，从 00 开始
- 秒：`s` 秒，从 0 开始，`ss` 秒，两位数，从 00 开始
- 毫秒：`S` 毫秒，从 0 开始，`SS` 毫秒，两位数，从 00 开始，`SSS` 毫秒，三位数，从 000 开始
- 时区：`Z` 时区与UTC的偏差，`ZZ` 时区与UTC的偏差，两位数
- AM/PM：`A` 上午或下午，AM或PM，`a` 上午或下午，am或pm
- 天数：`Do` 一个月中的第几天，如 1st、2nd、3rd 或 4th
- 时间戳：`X` Unix 秒时间戳，`x` Unix 毫秒时间戳

```javascript
dayjs().format('YYYY-MM-DD HH:mm:ss') // 2023-10-01 12:00:00
dayjs().format('YYYY-MM-DD HH:mm:ss.SSS') // 2023-10-01 12:00:00.000
dayjs().format('YYYY-MM-DD HH:mm:ss.SSS Z') // 2023-10-01 12:00:00.000 +08:00
dayjs().format('YYYY-MM-DD HH:mm:ss.SSS A') // 2023-10-01 12:00:00.000 PM
```

日期获取和设置

- 日期获取核心方法：`dayjs().get(unit)`，unit 为指定单位
- 日期设置核心方法：`dayjs().set(unit, value)`，unit 为指定单位，value 为指定值
- 快捷方法：`dayjs().get(unit) === dayjs()[unit]()`/`dayjs().set(unit, value) === dayjs()[unit](value)`
  - `dayjs().year()` 获取或设置年份
  - `dayjs().month()` 获取或设置月份，0-11
  - `dayjs().date()` 获取或设置日期，1-31
  - `dayjs().day()` 获取或设置星期几，周日到周六，0-6
  - `dayjs().hour()` 获取或设置小时
  - `dayjs().minute()` 获取或设置分钟
  - `dayjs().second()` 获取或设置秒
  - `dayjs().millisecond()` 获取或设置毫秒

```javascript
dayjs().get('year')
dayjs().set('year', 2000)
dayjs().get('month') // start 0
dayjs().set('month', 0) // start 0
dayjs().get('date')
dayjs().set('date', 1)
dayjs().get('day') // start 0
dayjs().set('day', 0) // start 0
dayjs().get('hour')
dayjs().set('hour', 12)
dayjs().get('minute')
dayjs().set('minute', 59)
dayjs().get('second')
dayjs().set('second', 1)
dayjs().get('millisecond')
dayjs().set('millisecond', 1)

dayjs().year() // gets current year
dayjs().year(2000) // returns new dayjs object

dayjs().month() // gets current month
dayjs().month(0) // returns new dayjs object

dayjs().date() // gets day of current month
dayjs().date(1) // returns new dayjs object

dayjs().day() // gets day of current week
dayjs().day(0) // returns new dayjs object

dayjs().hour() // gets current hour
dayjs().hour(12) // returns new dayjs object

dayjs().minute() // gets current minute
dayjs().minute(59) // returns new dayjs object

dayjs().second() // gets current second
dayjs().second(1) // returns new dayjs object

dayjs().millisecond() // gets current millisecond
dayjs().millisecond(1) // returns new dayjs object
```
