# Vue

## Vue 简介

Vue 是一个用于构建用户界面的渐进式 JavaScript 框架。它被设计为可以逐步采用，从简单的表单输入绑定到复杂的单页应用程序。

- Vue2 英文官网：https://v2.vuejs.org/
- Vue2 中文官网：https://v2.cn.vuejs.org/

> Vue2 已进入维护阶段，不再新增功能，文档单独托管：

Vue 历史

- 2013年，受到Angular框架的启发，尤雨溪开发出了一款轻量框架一Seed。同年12月，Seed更名为Vue，版本号0.6.0。
- 2014年，Vue正式对外发布，版本号0.8.0。Taylor otwell在Twitter上发表动态，说自己正在学习Vue.js。
- 2015年，10月27日，正式发布Vue 1.0.0 `Evangelion`（新世纪福音战士）。
- 2016年，10月1日，正式发布Vue 2.0.0 `Ghost in the Shell`（攻壳机动队）。
- 2020年，9月18日，正式发布Vue 3.0.0 `One Piece`（海贼王）。

Vue 特点

- 采用组件化模式，提高代码复用率及项目可维护性。
- 声明式编码，让编码人员无需直接操作DOM，提高开发效率。
- 使用虚拟DOM+优秀的Diff算法，尽量复用DOM节点。

## Vue 基础

### MVVM 模型

MVVM 模型

- M：模型（Model），对应 data 中的数据
- V：视图（View），对应模板代码
- VM：视图模型（ViewModel），对应 Vue 实例

### Vue 基础语法

创建根标签

```html
<div id="root">
  <!-- 正常书写html标签 -->
</div>
```

创建 Vue 实例

```javascript
// 直接创建
new Vue({ config })
// 先创建，再指定
const vm = new Vue({ config })
```

- `el`：element，指定要控制的区域
  - 直接在创建 Vue 实例时传入 `el: '#root'`
  - 也可以先创建 Vue 实例，再通过 `vm.$mount('#root')` 指定要控制的区域

```javascript
new Vue({
  el: '#root',
})
```

```javascript
// 先创建 Vue 实例，再通过 $mount 指定要控制的区域
const vm = new Vue({})
vm.$mount('#root')
// 也可以简写为
new Vue({}).$mount('#root')
```

- `data`：数据
  - 可以写对象，在创建 Vue 实例时传入 `data: {}`
  - 也可以写函数，在创建 Vue 实例时传入 `data: function() { return {} }`，可以简写为 `data() { return {} }`

```javascript
// 对象形式
new Vue({
  data: { name: '张三', age: 18 },
})
```

```javascript
// 函数形式
new Vue({
  data: function () {
    return { name: '张三', age: 18 }
  },
})
// 函数形式简写
new Vue({
  data() {
    return { name: '张三', age: 18 }
  },
})
```

- `methods`：方法
  - 以正常函数形式书写，this 指向 Vue 实例；以箭头函数形式书写，this 指向 window。所以建议使用正常函数形式书写，书写时可以省略 `function` 关键字，直接书写函数名和函数体即可
  - methods 中的函数，在使用时，可以直接获取一个事件对象 `event`，可以通过 `$event` 占位使用传递事件对象，其他参数正常传递，也可以不写 `$event`，直接默认传递事件对象 `event`

```javascript
new Vue({
  methods: {
    showInfo: function () {
      console.log(this.name) // this 指向 Vue 实例
      console.log(event) // 默认传入 event 为事件对象
    },
  },
})
// 简写
new Vue({
  methods: {
    showInfo() {
      console.log(this.name) // this 指向 Vue 实例
      console.log(event) // 默认传入 event 为事件对象
    },
  },
})
// 传参
new Vue({
  methods: {
    showInfo(name) {
      console.log(name) // 传入的参数
      console.log(event) // 默认传入 event 为事件对象
    },
  },
})
// 传参，通过 $event 获取事件对象
new Vue({
  methods: {
    showInfo(event, name) {
      console.log(event) // 传入的第一个参数，即$event占位的事件对象
      console.log(name) // 传入的第二个参数
    },
  },
})
```

- `computed`：计算属性
  - 计算属性会第一次调用时进行计算，将计算结果缓存起来，下次直接使用缓存结果，无需再次计算，节省性能。还会在其依赖的属性发生变化时，重新计算
  - 正常书写需要有 getter 和 setter，getter 负责获取计算结果，setter 负责修改计算结果
  - 当没有setter，只有getter时，计算属性可以进行简写：去掉 setter，将 getter 的内容直接放到函数体中即可

```javascript
//完整写法
new Vue({
  data: { name: '张三' },
  computed: {
    message: {
      get() {
        return `My name is ${this.name}`
      },
      set(value) {
        this.name = value
      },
    },
  },
})
// 简写
new Vue({
  data: { name: '张三' },
  computed: {
    message() {
      return `My name is ${this.name}`
    },
  },
})
```

- `watch`：侦听器

### Vue 模板语法

插值语法

- `{{ xxx }}`：xxx 是 JavaScript 表达式，可以读取到 data 中的所有属性，用于解析标签体内容

```html
<div id="root">
  <div>{{name}}，{{age}}</div>
  <div>{{name + '!'}}</div>
  <div>{{1 + 1}}</div>
  <div>{{age >= 18 ? '成年人' : '未成年人'}}</div>
</div>
```

指令语法

- `v-bind`：用于单向绑定属性值，`v-bind:xxx` 简写为 `:xxx`

```html
<div id="root">
  <!-- 完整写法 -->
  <a v-bind:href="url">百度</a>
  <!-- 简写 -->
  <a :href="url">百度</a>
</div>
```

- `v-model`：用于双向绑定属性值，控制表单元素的 value 属性，`v-model:value` 可简写为 `v-model`

```html
<div id="root">
  <!-- 完整写法 -->
  <input type="text" v-model:value="name" />
  <!-- 简写 -->
  <input type="text" v-model="name" />
</div>
```

- `v-on`：用于绑定事件，`v-on:click` 简写为 `@click`
  - 正常使用时，无需加括号，直接调用函数名即可。正常不传参时，会自动传递事件对象 `event`
  - 如果需要传参，可以正常传参，也可以通过 `$event` 来进行占位获取事件对象，其他参数正常传递，也可以不写 `$event`，直接默认传递事件对象 `event`

```html
<div id="root">
  <!-- 完整写法 -->
  <button v-on:click="showInfo">点我</button>
  <!-- 简写 -->
  <button @click="showInfo">点我</button>

  <!-- 传参 -->
  <button @click="showInfo('张三')">点我</button>
  <button @click="showInfo($event, '张三')">点我</button>
</div>
```

- 事件修饰符：用于修饰事件，例如阻止默认行为、阻止事件冒泡等
  - `@click.prevent`：阻止默认行为
  - `@click.stop`：阻止事件冒泡
  - `@click.once`：事件只触发一次
  - `@click.capture`：使用事件捕获模式
  - `@click.self`：只有当事件是从侦听器绑定的元素本身触发时才触发回调
  - `@click.passive`：事件的默认行为立即执行，无需等待事件回调执行完毕
- 按键修饰符：用于修饰键盘事件，例如按下回车键 keyup/keydown 时触发事件。
  - `@keyup.enter`：按下回车键时触发事件
  - `@keyup.tab`：按下 tab 键时触发事件
  - `@keyup.delete`：按下 delete 键时触发事件
  - `@keyup.esc`：按下 esc 键时触发事件
  - `@keyup.space`：按下空格键时触发事件
  - `@keyup.up`：按下上箭头键时触发事件
  - `@keyup.down`：按下下箭头键时触发事件
  - `@keyup.left`：按下左箭头键时触发事件
  - `@keyup.right`：按下右箭头键时触发事件
  - `@keyup.按键名称`：按下指定按键时触发事件
  - 有四个系统修饰键：ctrl、alt、shift、meta，可以组合使用，例如 `@keyup.ctrl.y`：按下 ctrl+y 按键时触发事件

### 数据代理与数据劫持

- 数据代理：通过一个对象代理另一个对象中属性的操作（读/写）
- 数据劫持：通过一个函数劫持另一个对象的属性操作（读/写）
