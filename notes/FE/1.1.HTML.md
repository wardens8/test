## HTML 简介

HTML 指的是超文本标记语言：**H**yper**T**ext **M**arkup **L**anguage，是用来描述网页的一种语言，文件扩展名为 `.html`。

HTML 不是一种编程语言，而是一种标记语言。标记语言是一套标记标签（markup tag）。HTML 使用标记标签来描述网页，HTML 文档包含了 HTML 标签及文本内容，HTML 文档也叫做 web 页面。

HTML 标签：HTML 标记标签通常被称为 HTML 标签 (HTML tag)。

- HTML 标签是由尖括号包围的关键词，比如 `<html>`
- HTML 标签通常是成对出现的，比如 `<b>` 和 `</b>`，标签对中的第一个标签是开始标签，第二个标签是结束标签，开始和结束标签也被称为开放标签和闭合标签
- HTML 标签也有一些标签是单独出现的，比如 `<br>`/`<br />` 标签就是单独出现的，在 HTML 文档中它叫作空标签（empty tag）。虽然没有强制规定，但是建议在空标签的末尾添加斜杠，以符合 XHTML 标准

```html
<标签>内容</标签>
<标签 />
```

HTML 元素：HTML 文档由 HTML 元素定义

- HTML 元素以开始标签起始，以结束标签终止，元素的内容是开始标签与结束标签之间的内容
- 某些 HTML 元素具有空内容（empty content），空元素在开始标签中进行关闭（以开始标签的结束而结束）
- 元素之间的关系：
  - 父元素：直接包含子元素的元素是父元素
  - 子元素：直接被父元素包含的元素
  - 祖先元素：直接或间接包含后代元素的元素是祖先元素，父元素也是祖先元素
  - 后代元素：直接或间接被祖先元素包含的元素是后代元素，子元素也是后代元素
  - 兄弟元素：拥有相同的父元素的元素是兄弟元素
- 大多数 HTML 元素可拥有属性
- HTML 标签和 HTML 元素通常都是描述同样的意思，但是严格来讲，一个 HTML 元素包含了开始标签与结束标签，如下实例：

```html
<p>这是一个段落。</p>
```

HTML 属性：属性是 HTML 元素提供的附加信息

- 属性通常出现在 HTML 标签的开始标签中，用于定义元素的行为、样式、内容或其他特性
- 属性总是以 `name="value"` 的形式写在标签内，`name` 是属性的名称，`value` 是属性的值
- 属性值应该始终被包括在引号内。双引号是最常用的，不过使用单引号也没有问题。

```html
<a href="http://www.runoob.com">这是一个链接</a>
```

HTML 注释：用来解释代码的信息，浏览器会忽略注释内容，注释内容不会显示在网页上

- 以 `<!--` 开始，以 `-->` 结束，比较长的注释可以在 `<!--` 和 `-->` 中分行写，长注释第一个字符缩进两个空格，更易于阅读
- 快捷键：`ctrl + /`

```html
<!-- 这是注释 -->

<!--
  这是一个较长注释。这是一个较长注释。这是一个较长注释。
  这是一个较长注释。这是一个较长注释。这是一个较长注释。
-->
```

Web 标准（蒂姆·伯纳斯·李 Tim Berners Lee）

- Web 标准是什么
  - Web 标准是由 W3C 组织和其他标准化组织制定的一系列标准的集合。W3C(万维网联盟）是国际最著名的标准化组织
- 为什么需要 Web 标准
  - 浏览器不同，它们显示页面或者排版就有些许差异。由于不同的浏览器解析出来的效果可能不一致，开发者常常要为多版本的开发而艰苦工作
  - 遵循 Web 标准除了可以让不同的开发人员写出的页面更标准、更统一外，还有以下优点：1.让 Web 的发展前景更广阔；2.内容能被更广泛的设备访问；3.更容易被搜寻引擎搜索；4.降低网站流量费用；5.使网站更易于维护；6.提高页面浏览速度
- Web 标准的构成
  - 主要包括结构（Structure）、表现（Presentation）和行为（Behavior）三个方面
  - 结构：结构用于对网页元素进行整理和分类，现阶段主要学的是 HTML
  - 表现：表现用于设置网页元素的版式、颜色、大小等外观样式，主要指的是 CSS
  - 行为：行为是指网页模型的定义及交互的编写，现阶段主要学的是 Javascript
  - Web 标准提出的最佳体验方案：结构、样式、行为相分离。简单理解：结构写到 HTML 文件中，表现写到 CSS 文件中，行为写到 JavaScript 文件中

## HTML 基础

基础结构

- `<!DOCTYPE html>` 文档类型声明，声明为 HTML5 文档
- `<html>` 元素是 HTML 页面的根元素，`lang="en"` 表示语言设置为英语，zh-CN 为中文
- `<head>` 元素表示网页的头部，包含了文档的元（meta）数据，如 `<meta charset="utf-8">` 定义网页编码格式为 utf-8
- `<body>` 元素表示网页的身体，包含了可见的页面内容
- `<title>` 元素描述了文档的标题

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Document</title>
  </head>
  <body></body>
</html>
```

meta 元数据标签：描述一些基本的元数据

```html
<!-- 为搜索引擎定义关键词 -->
<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript" />
<!-- 为网页定义描述内容 -->
<meta name="description" content="免费 Web & 编程 教程" />
<!-- 定义网页作者 -->
<meta name="author" content="Runoob" />
<!-- （必备）为网页定义编码格式 utf-8 -->
<meta charset="utf-8" />
<!-- 每30秒钟刷新当前页面 -->
<meta http-equiv="refresh" content="30" />
<!-- 控制 IE 浏览器渲染模式（仅 IE 生效） -->
<meta http-equiv="X-UA-Compatible" content="IE=edge" />
```

字符编码

- 网站编码格式是确保网页内容正确显示的关键设置，主要涉及字符编码标准、设置方法及优化建议
- 常见编码标准
  - `UTF-8`：支持全球所有语言字符，兼容 ASCII，是当前最广泛使用的编码格式
  - `GBK`/`GB2312`：主要用于中文网站，兼容 GB2312 扩展字符集
  - `ASCII`：仅支持英文字符（256 个），适用于纯英文内容

### 容器与布局标签

- 区块标签
  - `<div>`：定义文档中的分区或节（块级元素）
  - `<span>`：定义内联分区或节（行内元素）
  - 二者的区别：`<div>` 块级元素单独占一行，`<span>` 行内元素可以与其他元素在同一行

```html
<!-- 通用块级容器（布局核心） -->
<div>我是一个大区块</div>

<!-- 通用行内容器（文本包裹） -->
<span>我是一个小区块</span>
```

- 全局属性：所有 HTML 元素都可以使用的属性
  - `id` 为元素指定唯一的标识符
  - `class` 为元素指定一个或多个类名，用于 CSS 或 JavaScript 选择
  - `style` 直接在元素上应用 CSS 样式
  - `title` 为元素提供额外的提示信息，通常在鼠标悬停时显示

```html
<div id="header">This is the header</div>
```

```html
<p class="text highlight">This is a highlighted text.</p>
```

```html
<p style="color: blue; font-size: 14px;">This is a styled paragraph.</p>
```

```html
<abbr title="HyperText Markup Language">HTML</abbr>
```

### 链接和图像标签

- 链接标签
  - `href` 指定链接的地址，可以链接文本、图片、下载、锚点
  - `target` 定义链接的打开方式
    - `_blank` 在新窗口或新标签页中打开链接
    - `_self` 在当前窗口或标签页中打开链接（默认）
    - `_parent` 在父框架中打开链接
    - `_top` 在整个窗口中打开链接，取消任何框架
  - `download` 提示浏览器下载链接目标而不是导航到该目标
  - `title` 定义链接的额外信息，当鼠标悬停在链接上时显示的工具提示
  - `#id` 用于链接锚点，通常在同一页面中跳转到某个特定位置
  - `id` 用于指定元素的 id 名（CSS 中定义）
  - `class` 用于指定元素的类名（CSS 中定义）
  - `style` 直接在元素上定义 CSS 样式
  - 空链接的几种设置方式：
    - `href="#"` 导航到页面顶部，占位符，捕获点击事件
    - `href="javascript:void(0)"` 阻止默认行为，不刷新页面，配合 JS 使用
    - `href=""` 会提交表单或跳转到当前页面 URL（含参数）
    - `href="about:blank"` 新窗口打开空白页面

```html
<a href="https://www.runoob.com">这是一个链接</a>
```

```html
<a href="https://www.runoob.com" target="_blank">新窗口打开 Example</a>
```

```html
<a href="file.zip" download>下载文件</a>
```

```html
<a href="https://www.runoob.com" title="访问 runoob 网站">访问 runoob</a>
```

```html
<!-- 链接到页面中的某个部分 -->
<a href="#section1">跳转到第1部分</a>
<div id="section1">这是第1部分</div>
```

```html
<a href="https://www.runoob.com" id="external-link">我是链接</a>
<a href="https://www.runoob.com" class="external-link">我是链接</a>
<a href="https://www.runoob.com" style="color: red;">我是链接</a>
```

- 图像标签
  - `src`： 指定存储图像的地址
  - `alt`：为图像定义一串预备的可替换的文本
  - `title`：鼠标放在图像上，显示的文字
  - `width `/ `height`：设置图像的宽度和高度
  - `border`：设置图像的边框粗细

```html
<img src="pulpit.jpg" alt="Pulpit rock" title="Pulpit rock" />
```

- 路径
  - 相对路径：以引用文件所在位置为参考基础，而建立出的目录路径。`./` 表示当前目录，可以省略；`../` 表示上一级目录，可以多级
  - 绝对路径：是指目录下的绝对位置，直接到达目标位置，通常是从盘符开始的路径或者网络地址

```html
<!-- 同一级图像标签 -->
<img src="img.jpg" />
<!-- 下一级图像标签 -->
<img src="images/img.jpg" />
<!-- 上一级图像标签 -->
<img src="../img.jpg" />

<!-- 电脑内部的图像标签 -->
<img src="C:\电脑路径\img.jpg" />
<!-- 网站上面的图像标签 -->
<img src="http://www.itcast.cn/images/logo.gif" />
```

### 表格标签

- HTML 表格是一种用于展示结构化数据的标记语言元素。HTML 表格由 `<table></table>` 标签来定义。
  - 每个表格均有若干行（由 `<tr></tr>` 标签定义）
  - 每行被分割为若干单元格（由 `<td></td>` 标签定义）
  - 表格可以包含标题行（由 `<th></th>` 标签定义）
  - 定义表格的标题部分（由 `<thead></thead>` 标签定义）
  - 定义表格的主体部分（由 `<tbody></tbody>` 标签定义）
  - 定义表格的汇总部分（由 `<tfoot></tfoot>` 标签定义）
  - 注意：标题部分、主体部分、汇总部分的顺序可以随意，且页面不会颠倒位置

```html
<table>
  <thead>
    <tr>
      <th>列标题1</th>
      <th>列标题2</th>
      <th>列标题3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>行1，列1</td>
      <td>行1，列2</td>
      <td>行1，列3</td>
    </tr>
    <tr>
      <td>行2，列1</td>
      <td>行2，列2</td>
      <td>行2，列3</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>汇总行，列1</td>
      <td>汇总行，列2</td>
      <td>汇总行，列3</td>
    </tr>
  </tfoot>
</table>
```

- 表格属性
  - `border`：规定表格单元是否拥有边框，现在已不推荐使用，建议使用 CSS 来设置边框
  - `align`：规定表格相对周围元素的对齐方式，left / center / right
  - `cellpadding`：规定单元边沿与其内容之间的空白，单位为像素
  - `cellspacing`：规定单元格之间的空白，单位为像素
  - `width `/ `height`：设置表格的宽度和高度
  - `colspan`：规定横向合并的单元格数量，合并列（横向合并）
  - `rowspan`：规定纵向合并的单元格数量，合并行（纵向合并）

```html
<table border="1" align="center" cellpadding="10" cellspacing="10"></table>
```

### 表单标签

- form 标签
  - `action` 指定接收并处理表单数据的服务器程序的 url 地址
  - `method` 指定表单数据的提交方式，其取值为 get 或 post
  - `name` 指定表单的名称，以区分同一个页面中的多个表单

```html
<form action="/" method="post" name="list"></form>
```

- 表单标签
  - `input` 输入框
    - `type` 规定输入框的类型
    - `name` 规定输入框的名字，单选框和复选框需要有相同的名字
    - `value` 规定输入框的默认值
    - `checked` 规定输入框是否选中，布尔属性
    - `disabled` 规定输入框是否可用，布尔属性
    - `readonly` 规定输入框是否只读，布尔属性
    - `required` 规定输入框是否必填，布尔属性
    - `placeholder` 规定输入框的提示信息
    - `maxlength` 规定输入框可以填入的最大长度
    - 布尔属性：布尔属性是指不需要值的属性，它们的存在即表示 true，不存在则表示 false
  - `label` 文字标注标签
    - `for` 指向输入框的 id
  - `select`和 `option` 下拉框和选择框
    - `name` 规定下拉框的名字
    - `value` 规定选择框的默认值
    - `selected` 规定选择框是否选中，布尔属性
  - `textarea` 文本域
    - `rows` 规定文本域的行数
    - `cols` 规定文本域的列数
  - `button` 按钮
    - `type` 规定按钮的类型
    - `value` 规定按钮的默认值

```html
<!-- 文本输入框 -->
<label for="name">用户名:</label>
<input type="text" id="name" name="name" disabled />
<input type="text" id="name" name="name" required />

<!-- 密码输入框 -->
```

## HTML 头部标签补充

head 元素包含了所有的头部标签元素。在 head 元素中你可以插入脚本（scripts）, 样式文件（CSS），及各种 meta 信息。可以添加在头部区域的元素标签为：`<title>`，`<meta>`，`<style>`，`<link>`，`<script>`，`<noscript>` 和 `<base>`

- `<style>`：定义 HTML 文档的样式文件引用地址，在 style 元素中你也可以直接添加样式来渲染 HTML 文档

```html
<style type="text/css">
  body {
    background-color: red;
  }
  p {
    color: blue;
  }
</style>
```

- `<link>`：定义文档与外部资源之间的关系，通常用于链接到样式表

```html
<link rel="stylesheet" type="text/css" href="index.css" />
```

- `<script>`：用于加载脚本文件或写入脚本，如 JavaScript

```html
<!-- 引入外部脚本文件 -->
<script src="index.js"></script>
<!-- 直接在页面中写入脚本 -->
<script type="text/javascript">
  console.log('Hello, World!')
</script>
```

- `<noscript>`：提供无法使用脚本时的替代内容，比方在浏览器禁用脚本时，或浏览器不支持客户端脚本时。元素内可包含普通 HTML 页面的 body 元素中能够找到的所有元素。只有在浏览器不支持脚本或者禁用脚本时，才会显示 noscript 元素中的内容

```html
<noscript>抱歉，您的浏览器不支持 JavaScript!</noscript>
```

- `<base>`：描述了基本的链接地址/链接目标，该标签作为 HTML 文档中所有的链接标签的默认链接

```html
<base href="http://www.runoob.com/images/" target="_blank" />
```

- `<meta>` 移动端适配补充
  - 适用场景：仅对移动设备浏览器生效，PC 端浏览器会直接忽略该标签
  - `name="viewport"` 表示布局视口=视觉视口
  - `content=""` 里面的内容
    - `width=device-width` 定义布局视口宽度：`device-width`（自动匹配手机屏幕宽度）
    - `initial-scale=1.0` 页面初始缩放比例：1.0=不缩放，和 `width=device-width` 搭配使用，双重保障
    - `maximum-scale=1.0` 页面最大允许缩放比例：禁止用户放大页面，避免适配错乱
    - `minimum-scale=1.0` 页面最小允许缩放比例：禁止用户缩小页面
    - `user-scalable=no` 是否允许用户手动缩放：移动端适配一般写 no，禁止缩放

```html
<!-- 移动端适配 标准meta视口标签，直接复制使用 -->
<meta
  name="viewport"
  content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=no"
/>
```

SEO 优化：TDK（Title、Description、Keywords）

- SEO（Search Engine Optimization）汉译为搜索引擎优化，是一种利用搜索引擎的规则提高网站在有关搜索引擎内自然排名的方式。SEO 的目的是对网站进行深度的优化，从而帮助网站获取免费的流量，进而在搜索引擎上提升网站的排名，提高网站的知名度。页面必须有三个标签用来符合 SEO 优化。
- `title`：核心词放前面，控制 30 字内，不重复，带卖点/品牌，不堆砌
- `description`：100 字内概括页面价值，加关键词和引导词（如 “立即咨询”），吸引点击
- `keywords`：选 3-5 个核心词，逗号分隔，和内容匹配，不用纠结排名作用

## HTML5 Video / Audio

- Video / Audio 属性
  - `src`：要播放的视频或音频的 URL
  - `width`：设置视频播放器的宽度（Audio 没有）
  - `height`：设置视频播放器的高度（Audio 没有）
  - `controls`：如果出现该属性，则向用户显示控件，比如播放按钮
  - `autoplay`：如果出现该属性，则视频或音频在就绪后马上播放
  - `loop`：如果出现该属性，则当媒介文件完成播放后再次开始播放
  - `muted`：如果出现该属性，视频或音频输出为静音
  - `poster`：规定视频正在下载时显示的图像，直到用户点击播放按钮（Audio 没有）
  - `preload`：auto / metadata / none：如果出现该属性，则视频或音频在页面加载时进行加载，并预备播放。如果使用 "autoplay"，则忽略该属性

```html
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4" />
  <source src="movie.ogg" type="video/ogg" />
  <source src="movie.webm" type="video/webm" />
  您的浏览器不支持 video 标签。
</video>

<audio controls>
  <source src="horse.mp3" type="audio/mpeg" />
  <source src="horse.ogg" type="audio/ogg" />
  <source src="horse.wav" type="audio/wav" />
  您的浏览器不支持 audio 元素。
</audio>
```

## HTML5 新表单元素

注意：不是所有的浏览器都支持 HTML5 新的表单元素，但是你可以在使用它们，即使浏览器不支持表单属性，仍然可以显示为常规的表单元素。

- `<datalist>` 元素规定输入域的选项列表。`<datalist>` 属性规定 form 或 input 域应该拥有自动完成功能。当用户在自动完成域中开始输入时，浏览器应该在该域中显示填写的选项：使用 `<input>` 元素的列表属性与 `<datalist>` 元素绑定


```html
<form oninput="x.value=parseInt(a.value)+parseInt(b.value)">

  <output name="x" for="a b"></output>

```

## HTML5 新表单属性

简单介绍常用的属性，其他的自行查询

- form 新属性
  - `autocomplete`：自动完成功能
  - `novalidate`：该属性是一个布尔属性。novalidate 属性是 HTML 表单元素的一个布尔属性，用于设置浏览器不对表单进行验证
- input 新属性
  - `autocomplete`：自动完成功能
  - `autofocus`：该属性是一个布尔属性。规定在页面加载时，域自动地获得焦点
  - `form`：规定输入域所属的一个或多个表单。

## HTML5 Web 存储

HTML5 提供了两种在客户端存储数据的新方法：

- `window.localStorage` 用于长久保存整个网站的数据，保存的数据没有过期时间，直到手动去除
- `window.sessionStorage` 用于临时保存同一窗口（或标签页）的数据，在关闭窗口或标签页之后将会删除这些数据

之前，这些都是由 cookie 完成的。但是 cookie 不适合大量数据的存储，因为它们由每个对服务器的请求来传递，这使得 cookie 速度很慢而且效率不高。这两种方法会在后续学习中补充。

# HTML(5) 代码规范

**约定优于配置**

1. 使用正确的文档类型
2. 使用小写元素名，不推荐使用大写、首字母大写、大小写混合使用（HTML5 元素名可以使用大写和小写字母）
3. 关闭所有 HTML 元素以及空的 HTML 元素（在 HTML5 中，你不一定要关闭所有元素，例如 `<p>` 元素，但我们建议每个元素都要添加关闭标签）
4. 使用小写属性名（HTML5 属性名允许使用大写和小写字母）
5. 属性值推荐使用引号（HTML5 属性值可以不用引号）
6. 图片属性通常使用 alt 属性。在图片不能显示时，它能替代图片显示。定义好图片的尺寸，在加载时可以预留指定空间，减少闪烁
7. 空格和等号（标签中的等号前后可以使用空格，但我们推荐少用空格）
8. 避免一行代码过长：使用 HTML 编辑器，左右滚动代码是不方便的。每行代码尽量少于 80 个字符
9. 空行和缩进：不要无缘无故添加空行。为每个逻辑功能块添加空行，这样更易于阅读。缩进使用两个空格，不建议使用 TAB。比较短的代码间不要使用不必要的空行和缩进
10. 不要省略 `<html>` 和 `<body>`（在标准 HTML5 中，`<html>` 和 `<body>` 标签是可以省略的）
11. 不要省略 `<head>`（在标准 HTML5 中，`<head>`标签是可以省略的。默认情况下，浏览器会将 `<body>` 之前的内容添加到一个默认的 `<head>` 元素上）
12. 元数据（HTML5 中 `<title>` 元素是必须的，标题名描述了页面的主题，标题和语言可以让搜索引擎很快了解你页面的主题）
13. CSS和JS优先从外部引入
14. 使用小写文件名：大多Web服务器（Apache, Unix）对大小写敏感，部分Web服务器（Microsoft, IIS）对大小写不敏感。你必须保持统一的风格，我们建议统一使用小写的文件名
15. 说到最后，代码规范我们会用统一安装插件来进行解决，上述内容了解即可
    - ESLint 插件
    - Prettier 插件
