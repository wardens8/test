# CSS 框架

CSS 框架是一组预定义的 CSS 类，用于快速开发网页。它提供了一套完整的 CSS 样式，包括布局、字体、颜色、按钮、表单等，可以帮助开发者快速构建网页，提高开发效率。常见的 CSS 框架有 Bootstrap、LayUI、TailwindCSS 等。

# Bootstrap

## Bootstrap 简介

- Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。
- Bootstrap 官网：https://getbootstrap.com/
- Bootstrap 中文网：https://www.bootcss.com/
- Bootstrap 中文手册：https://www.bootstraph.com/
- Bootstrap 目前有三个主要版本：v3、v4、v5。各网站都能查询到各个版本的使用文档。
- 注意：
  - Bootstrap3 支持 IE8+ 浏览器，依赖 jQuery，使用前需先引入 jQuery 库
  - Bootstrap4 支持 IE10+ 浏览器，不依赖 jQuery
  - Bootstrap v5 不支持 IE 浏览器，不依赖 jQuery

> 本文内容按照 Bootstrap v3 版本进行介绍，如需使用其他新版本，请自行查阅对应版本的使用文档，内容大致相同，只是部分类名、样式、功能有所变化，文本中不再赘述

## Bootstrap 安装

- 官网下载本地安装

```html
<link rel="stylesheet" href="bootstrap/css/bootstrap.min.css" />
<script src="bootstrap/js/bootstrap.min.js"></script>
```

- CDN 安装

```html
<!-- bootstrap v5 -->
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css"
/>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js"></script>
```

```html
<!-- bootstrap v4 -->
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/css/bootstrap.min.css"
/>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/js/bootstrap.bundle.min.js"></script>
```

```html
<!-- bootstrap v3 -->
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@3.4.1/dist/css/bootstrap.min.css"
/>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@3.4.1/dist/js/bootstrap.min.js"></script>
```

```html
<!-- bootstrap v3 -->
<link
  rel="stylesheet"
  href="https://cdn.bootcdn.net/ajax/libs/twitter-bootstrap/3.4.1/css/bootstrap.min.css"
```

- npm 安装

```bash
npm install bootstrap
npm install bootstrap@version

yarn add bootstrap
yarn add bootstrap@version

pnpm add bootstrap
pnpm add bootstrap@version
```

- 开始使用

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Bootstrap</title>
    <link rel="stylesheet" href="bootstrap/css/bootstrap.min.css" />
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <script src="bootstrap/js/bootstrap.min.js"></script>
  </body>
</html>
```

## Bootstrap 基础

### Bootstrap 布局

- `container` 类用于固定宽度并支持响应式布局的容器
- `container-fluid` 类用于 100% 宽度，占据全部视口（viewport）的容器

```html
<!-- .container 类用于固定宽度并支持响应式布局的容器。 -->
<div class="container">...</div>

<!-- .container-fluid 类用于 100% 宽度，占据全部视口（viewport）的容器。 -->
<div class="container-fluid">...</div>
```

### Bootstrap 栅格系统

- 简单使用

```html
<div class="container">
  <div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-8">.col-md-8</div>
  </div>
  <div class="row">
    <div class="col-md-8">.col-md-8</div>
    <div class="col-md-4">.col-md-4</div>
  </div>
</div>
```

- `row` 类用于创建水平方向的列容器
- `col-*-*` 类用于创建列，其中 `*` 表示屏幕大小，`*` 表示占据的列数，`*` 表示占据的列数
  - `col-xs-*` 类用于创建超小屏幕（手机）的列
  - `col-sm-*` 类用于创建小屏幕（平板）的列
  - `col-md-*` 类用于创建中等屏幕（桌面显示器）的列
  - `col-lg-*` 类用于创建大屏幕（大桌面显示器）的列

```html
<div class="container">
  <div class="row">
    <div class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
      .col-xs-12 .col-sm-6 .col-md-4 .col-lg-3
    </div>
    <div class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
      .col-xs-12 .col-sm-6 .col-md-4 .col-lg-3
    </div>
    <div class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
      .col-xs-12 .col-sm-6 .col-md-4 .col-lg-3
    </div>
    <div class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
      .col-xs-12 .col-sm-6 .col-md-4 .col-lg-3
    </div>
  </div>
</div>
```

- 列偏移
  - `col-md-offset-*` 类用于创建列偏移，其中 `*` 表示偏移的列数

```html
<div class="container">
  <div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 col-md-offset-4">.col-md-4 .col-md-offset-4</div>
  </div>
</div>
```

- 列嵌套

```html
<div class="container">
  <div class="row">
    <div class="col-md-8">
      <div class="row">
        <div class="col-md-6">.col-md-6</div>
        <div class="col-md-6">.col-md-6</div>
      </div>
    </div>
    <div class="col-md-4">.col-md-4</div>
  </div>
</div>
```

- 列排序
  - `col-md-push-*` 类用于创建列排序（向后），其中 `*` 表示排序的列数
  - `col-md-pull-*` 类用于创建列排序（向前），其中 `*` 表示排序的列数

```html
<div class="container">
  <div class="row">
    <div class="col-md-9">.col-md-9</div>
    <div class="col-md-3">.col-md-3</div>
  </div>
  <div class="row">
    <div class="col-md-3 col-md-push-9">.col-md-3 .col-md-push-9</div>
    <div class="col-md-9 col-md-pull-3">.col-md-9 .col-md-pull-3</div>
  </div>
</div>
```

### Bootstrap 按钮

- 简单使用

```html
<a class="btn btn-default" href="#" role="button">Link</a>
<button class="btn btn-default" type="submit">Button</button>
<input class="btn btn-default" type="button" value="Input" />
<input class="btn btn-default" type="submit" value="Submit" />
```

- `.btn` 基础样式
- `.btn-default` 默认样式
- `.btn-primary` 首选项样式
- `.btn-success` 成功样式
- `.btn-info` 一般信息样式
- `.btn-warning` 警告样式
- `.btn-danger` 危险样式
- `.btn-link` 链接样式

```html
<!-- 默认样式 -->
<button class="btn btn-default">Default</button>
<!-- 首选项样式 -->
<button class="btn btn-primary">Primary</button>
<!-- 成功样式 -->
<button class="btn btn-success">Success</button>
<!-- 一般信息样式 -->
<button class="btn btn-info">Info</button>
<!-- 警告样式 -->
<button class="btn btn-warning">Warning</button>
<!-- 危险样式 -->
<button class="btn btn-danger">Danger</button>
<!-- 链接样式 -->
<button class="btn btn-link">Link</button>
```

- `.btn-lg` 大尺寸
- `.btn-sm` 小尺寸
- `.btn-xs` 超小尺寸

```html
<!-- 大尺寸 -->
<button class="btn btn-primary btn-lg">Large button</button>
<!-- 默认尺寸 -->
<button class="btn btn-primary">Default button</button>
<!-- 小尺寸 -->
<button class="btn btn-primary btn-sm">Small button</button>
<!-- 超小尺寸 -->
<button class="btn btn-primary btn-xs">Extra small button</button>
```

- `.btn-block` 块级按钮

```html
<button class="btn btn-primary btn-block">Block level button</button>
```

- `.active` 激活状态

```html
<button class="btn btn-primary active">Primary button</button>

<a href="#" class="btn btn-primary active" role="button">Primary link</a>
```

- `disabled` 禁用状态

```html
<button class="btn btn-primary" disabled="disabled">Primary button</button>
<button class="btn btn-default" disabled="disabled">Button</button>

<a href="#" class="btn btn-primary  disabled" role="button">Primary link</a>
<a href="#" class="btn btn-default  disabled" role="button">Link</a>
```

# LayUI

- LayUI 是一款由职业前端倾情打造，面向全栈开发的前端 UI 框架，其自底向上地封装，遵循 HTML5 标准的 CSS 框架，从核心代码到 API 的每一处细节都经过精心雕琢，遵循设计原则，让前端开发如丝般顺滑。
- LayUI 官网：https://layui.dev/
- 具体内容与 Bootstrap 大同小异，自行查看文档，不再赘述

# TailwindCSS

- Tailwind CSS 是一个用于快速构建自定义用户界面的实用工具优先的 CSS 框架。它由 Adam Wathan 创建，旨在帮助开发者更快地构建自定义设计。
- Tailwind CSS 官网：https://tailwindcss.com/
- Tailwind CSS 中文网：https://www.tailwindcss.cn/

# WeUI

- WeUI 是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信内网页和微信小程序量身设计，令用户的使用感知更加统一。
- WeUI 官网：https://weui.io/
