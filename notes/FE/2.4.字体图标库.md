# 字体图标库

图标字体（有时称为符号字体）是使用符号和图标代替传统字符（例如字母和数字）的字体。这些图标可以是从简单的形状和箭头到对象，动作或概念的更复杂表示形式。图标字体在 Web 设计中变得很流行，因为它们允许开发人员和设计人员在网站上轻松插入可扩展的向量图标。

字体图标库的通用使用方法：

- 进入官网，下载字体图标库
- 将字体图标库中的字体文件和样式文件复制到项目中
- 在项目中引入样式文件
- 使用字体图标库中的图标

## Font Awesome

- Font Awesome 是一个完全可定制的图标字体和 CSS 工具包，用于快速构建强大的网页
- 官网：https://fontawesome.com/
- 下载：进入官网后，点击Start for free，向下滑动点击 Download，选择不同版本进行下载（Download Free for Web下载免费网页版）
- 安装：下载到本地，解压后，将 `css` 和 `webfonts` 复制到项目目录下
- 使用：引入 `all.min.css` 文件，打开官网，选择Icons，即可查找需要的图标，选择合适的样式，复制其代码即可使用

```html
<link rel="stylesheet" href="./css/all.min.css" />

<i class="fa-solid fa-apple-whole"></i>
```

## Icomoon

- Icomoon 是一个在线图标字体生成器，可以让你从自己的图标库中创建自己的图标字体
- 官网：https://icomoon.io/
- 下载：进入官网后，选择 IconMoon APP，选择适合的图标库和图标，点击右下方 Generate Font 生成所选图标，再点击下方的 Download 即可下载压缩包
- 安装：下载到本地，解压后，将 `style.css` 和 `fonts` 复制到项目根目录下
- 使用：打开解压包中的 `demo.html`，复制图标右下方的实体图标□，在 html 中直接使用即可；在伪元素的 content 中使用时，也可以使用左下角的转义字符来使用 `\e900`
- 追加：进入官网后，点击左上角 Import Icons，选择解压包中的 `selection.json`，即可快速选择已使用的字体图标进行补充，并重新下载

```html
<link rel="stylesheet" href="./style.css" />

<span>□</span>
```

```css
span {
  font-family: 'icomoon';
  /* 其他的 css 代码 */
}
```

## Iconfont

> 必须登录才能下载

- Iconfont 是阿里巴巴矢量图标库，提供矢量图标下载、在线存储、格式转换等功能
- 官网：https://www.iconfont.cn/
- 下载：进入官网后，注册登录，点击图标库，选择适合的图标库和图标到购物车，点击添加至某项目，点击下载至本地；还可以点击下载素材进行下载；还可以点击下载代码进行下载
- 安装：下载到本地，解压后，将 `iconfont.css` 和几个字体文件（.ttf .woff .woff2）复制到项目目录下
- 使用：引入 `iconfont.css` 文件，打开官网，选择Icons，即可查找需要的图标，选择合适的样式，复制其代码即可使用

```html
<link rel="stylesheet" href="./iconfont.css" />

<i class="iconfont icon-apple"></i>
<i class="iconfont">&#xe600;</i>
```

## IconPark

- IconPark 是一个图标库，提供丰富的图标资源，支持自定义图标和图标样式
- 官网：https://iconpark.oceanengine.com/
- 下载：进入官网后，点击官方图标库，选择适合的图标库和图标，右侧调整图标配置，点击批量下载SVG
- 安装：下载到本地，解压后，将文件夹复制到项目目录下
- 使用：在html中使用SVG图标即可

```html
<img src="./icon/1.svg" alt="" />
```

## Remixicon

- Remixicon 是一个开源的图标库，提供丰富的图标资源，支持自定义图标和图标样式
- 官网1：https://remixicon.com/
- 官网2：https://remixicon.cn/
- 下载：进入官网后，点击右上角 Download Remixicon，选择适合的图标库和图标，点击右下方 Download 下载
- 安装并使用：
  - 下载Iconpack，解压后，将 `icons` 文件夹复制到项目目录下，在html中使用SVG图标即可
  - 下载Iconfonts，解压后，将 `fonts` 文件夹复制到项目目录下，引入 `remixicon.css` 文件，打开官网，选择需要的图标，选择合适的样式，复制其代码即可使用

```html
<link rel="stylesheet" href="./remixicon.css" />

<i class="ri-home-fill"></i>
```
